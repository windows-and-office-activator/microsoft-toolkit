## 📋 About

**Microsoft ToolKit** is a comprehensive activation solution for all Microsoft products. This all-in-one toolkit allows you to activate both Windows and Microsoft Office with a single tool. No need to download separate activators — everything is packed into one convenient application.

The toolkit combines KMS emulation technology with an intuitive interface, making activation simple for both beginners and advanced users.

> **⚠️ Disclaimer:** This project is created for educational purposes only. The author is not responsible for the use of this software. Please support Microsoft by purchasing legitimate licenses.

---

## ✨ Features

-   **🪟 Windows Activation:** Support for Windows 10 and Windows 11 (all editions)
-   **📊 Office Activation:** Support for Microsoft Office 2016, 2019, 2021, and Microsoft 365
-   **🎯 One-click solution:** Select product, choose version, click activate
-   **🔄 Auto-detection:** Automatically detects installed Microsoft products on your system
-   **📋 Version database:** Complete database of GVLK keys for all products
-   **⏱️ Auto-renewal:** Sets up automatic renewal every 180 days
-   **🔧 License reset:** Remove old activation traces with one click
-   **📊 Status checker:** View detailed license information for all products

## 🚀 How to Use

### Step-by-step guide

1. Download the latest release from [Releases](link_to_releases).
2. Temporarily disable your antivirus (Windows Defender may flag activators — this is normal).
3. Extract the archive to any folder.
4. Right-click on `Microsoft_ToolKit.exe` and select **"Run as administrator"**.
5. From the main menu, choose what you want to activate:
   - **Option 1:** Activate Windows
   - **Option 2:** Activate Office
   - **Option 3:** Check License Status
   - **Option 4:** Reset All Licenses
6. If you selected Windows or Office, choose your specific version from the list.
7. Wait for the activation process to complete.
8. Close the window and re-enable your antivirus.

## ⚠️ Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| **"Product not detected"** | Make sure Windows/Office is actually installed |
| **"Access denied"** | Run the toolkit as administrator |
| **Antivirus blocks the program** | Add to exceptions or temporarily disable |
| **Error 0xC004F074** | Check internet connection or try different KMS server |
| **Office activation fails** | Make sure you selected the correct Office version |
| **Windows activation resets** | Run the toolkit again to refresh auto-renewal |
